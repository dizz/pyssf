Welcome to Service Sharing Facility's documentation!
====================================================

:Release: |version|
:Date: |today|

.. image:: images/pyssf_logo.png
   :alt: Service Sharing Facility logo

.. note::
    This is still work in progress. For more information and tracking the
    ongoing developement go to http://sourceforge.net/projects/pyssf/

This software package is a collection of modules which form an uniform
integration point for Grid/Cloud/Cluster products. If provides several
front-ends which can easily be used to build complete Services or integrate
products with your products. It aims to deliver an distributed ecosystem where
multiple Grid/Cloud/Clusters sites can federate. It is developed under a LGPL
license.

Features:
 * Access multiple Distributed Resource Managers (DRM) systems
 * Provides simple Cloud interfaces for (Infrastructure|Platform) as a Service
   (Job submission, VM management, and many more...)
 * Negotiation and federation of services between Grids/Clouds/Clusters
 * Integration point to integrate your products with Grids/Clouds/Clusters
